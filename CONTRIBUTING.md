# Contributing Guidelines

Thank you for your interest in contributing to the Ayurveda Plant Detection and Suggestions project! We welcome contributions from everyone. Please follow the guidelines below to ensure a smooth contribution process.

## Contribution Guidelines
- Please ensure that your contributions align with the project guidelines and objectives.
- Discuss any large changes or enhancements in an issue before you start working on them.
- Keep your pull requests focused and concise.

## Development Setup
To set up the development environment, follow these steps:

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/LeviAckereman/Ayurveda-Plant-Detection-and-Suggestions.git
   cd Ayurveda-Plant-Detection-and-Suggestions
   ```
2. **Install necessary dependencies:**  
   ```bash
   npm install
   ```
   (or the equivalent command for your environment)
3. **Run the project:**  
   ```bash
   npm start
   ```

## Pull Request Process
1. **Fork the Repository**: Use the GitHub interface to fork the repository.
2. **Create a Branch**: Create a branch for your feature or bugfix. 
   ```bash
   git checkout -b your-feature-name
   ```
3. **Make Your Changes**: Develop your changes, making sure to follow the code style guidelines.
4. **Commit Your Changes**:  
   ```bash
   git add .
   git commit -m "Your detailed description of your changes"
   ```
5. **Push to GitHub**:  
   ```bash
   git push origin your-feature-name
   ```
6. **Create a Pull Request**: Go to the original repository and select the `Pull Requests` tab, click on `New Pull Request`, and follow the instructions.

Thank you for contributing!
